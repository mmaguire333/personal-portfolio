About Me

Hi there, I'm Matt. I'm a new front-end developer from Colorado. Initially, I was drawn to web development and coding in general through my love for building things. My learning journey up to this point has been a creative outlet that has really excited me. Another thing that drew me to web development is the enjoyment I get out of solving puzzles. Whether its a rubiks cube, a chess puzzle, or working through a complex program I love using the logical side of my brain to solve challenging problems.

My educational background is in mathematics. In December 2021 I graduated with a BS in Mathematics. What I loved about math was the way in which abstract and seemingly unrelated ideas could be brought together and used to solve challenging problems. As I've been learning web development its been fun to see how this type of reasoning can be also used to solve problems with a programming language like javascript. The similarties between thinking like a mathematician and thinking like a programmer has been something that I've enjoyed discovering.

Overall, I have really enjoyed my learning journey so far, and I can't wait to see where it will take me next.

The Game of Life

The Game of Life is a game invented by mathematician John Conway. It is played on a grid where the user can fill in squares by clicking on them. Once the game is started the grid squares evolve according to a simple set of related to the state of each square. It is an interesting example of complex structure being produced by simple rules.

Weather App

This is a simple weather app in which the user can search for a city and view relevant weather data. This app displays current data for temperature, feels like temperature, windspeed, and description. It also has a five day, three hour forecast that displays predicted temperature, feels like temperature, precipitation chance, and description.

Tic Tac Toe

This project is an implementation of the game tic tac toe. The user can play against another human or play against the computer. Playing the computer has three difficulty levels. On easy the computer plays randomly, on medium the computer will try to block the user from winning, and on hard the computer plays optimally.
